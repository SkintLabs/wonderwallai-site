Hey Product Hunt.

I'm Louis, solo founder of Skint Labs. Today I'm launching WonderwallAi, an open-source AI firewall SDK I built because LLM apps are way too easy to abuse.

The problem... prompt injection, data leaks, off-topic abuse. Once you ship an LLM product bad actors find ways in fast.

WonderwallAi blocks prompt injection before it reaches your LLM, catches API keys and PII leaking in responses, enforces topic boundaries to stop jailbreaking, and sanitises file uploads stripping EXIF and GPS metadata. It runs locally with zero external API calls and under 2ms latency.

It's MIT licensed. Run "pip install wonderwallai" and you're protected in 3 lines of code.
